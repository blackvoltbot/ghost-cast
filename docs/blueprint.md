# **App Name**: GhostCast

## Core Features:

- Secure Admin Terminal: A centralized login portal providing access to the administrative remote support dashboard with multi-factor authentication capability.
- GhostLink Generator: Creation of unique, ephemeral room IDs for one-time secure remote support sessions.
- WebRTC Screen Transmission: Seamless, high-performance screen sharing using WebRTC to allow admins to view user desktops in real-time without plugins.
- Bi-directional Voice Link: Instant two-way audio communication between the admin and the user to facilitate spoken guidance.
- Session Monitoring Engine: Real-time room status tracking with indicators for connection health, user presence, and media activity.
- Real-time Signaling Bridge: A high-concurrency WebSocket server implementation to manage session handshakes and metadata.
- Incident Diagnosis Tool: A generative AI-powered tool that analyzes session metadata and console logs to suggest immediate technical fixes to the support admin.

## Style Guidelines:

- Primary color: High-intensity Ruby Red (#FF3131) for high-impact alerts and key functional buttons.
- Background color: Deep Onyx (#0D0909), a desaturated dark charcoal with subtle red undertones to anchor the hacker aesthetic.
- Accent color: Vibrant Crimson (#FF5E5E) used for session indicators and active hover states to provide stark contrast.
- Headline and navigation font: 'Space Grotesk', a geometric sans-serif that delivers a high-tech, digital-first impression.
- Data and Code display: 'Source Code Pro', used for room links and terminal output to maintain the monospace 'hacker' feeling.
- The interface features a strictly 'Command Center' layout with sharp corners, glowing borders, and grid-based module placement for a precise, tactical look.
- Scanning effects and pulse animations for the status indicators to evoke a sense of continuous live monitoring.
